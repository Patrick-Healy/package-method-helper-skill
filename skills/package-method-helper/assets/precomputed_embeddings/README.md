# Precomputed Embeddings Contract

This directory is reserved for published embedding assets that correspond to the safe chunk export generated by `scripts/export_embedding_chunks.py`.

## Expected files

Recommended shape:
- `embeddings_manifest.json`
- `chunk_embeddings.parquet`
- optional split files such as `chunk_embeddings_r.parquet`, `chunk_embeddings_python.parquet`, `chunk_embeddings_stata.parquet`

## Required manifest fields

- `schema_version`
- `source_db_digest`
- `source_chunk_export_digest`
- `embedding_model`
- `embedding_dimensions`
- `created_utc`
- `record_count`
- `languages`
- `notes`

## Required vector columns

- `id`
- `language`
- `package`
- `canonical_package_name`
- `doc_type`
- `chunk_type`
- `importance_tier`
- `embedding`

## Safety constraints

- no local filesystem paths
- no usernames
- no internal scratch metadata
- embeddings must line up with the safe JSONL export IDs
